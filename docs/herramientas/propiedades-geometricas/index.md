# Propiedades Geométricas de Secciones

Referencia de propiedades geométricas para secciones transversales de uso común en ingeniería estructural: centroides, momentos de inercia, módulos elásticos de sección y radios de giro.

## Propiedades de interés

| Propiedad | Símbolo | Descripción |
|---|---|---|
| Área | $A$ | Área de la sección transversal |
| Centroide | $\bar{y}$, $\bar{z}$ | Coordenadas del centro geométrico |
| Momento de inercia | $I_x$, $I_y$ | Segundo momento de área respecto a cada eje |
| Momento de inercia polar | $J$ | $I_x + I_y$ |
| Módulo elástico de sección | $S = I / c$ | Relación entre inercia y fibra extrema |
| Radio de giro | $r = \sqrt{I/A}$ | Medida de distribución del área |
| Momento de inercia producto | $I_{xy}$ | Para ejes no principales |
